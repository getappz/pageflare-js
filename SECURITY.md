# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in pageflare, please report it responsibly.

**Do not open a public issue.** Instead, email **security@pageflare.dev** with:

- A description of the vulnerability
- Steps to reproduce
- Impact assessment (what an attacker could do)

We will acknowledge your report within 48 hours and aim to provide a fix within 7 days for critical issues.

## Scope

This policy covers:

- The `pageflare` npm package and its framework integrations
- The `@pageflare/cli` npm package and its platform-specific binaries
- The pageflare.dev website and API

## Supported Versions

We only patch the latest release. Please upgrade to the latest version before reporting.

| Version | Supported |
|---------|-----------|
| Latest  | Yes       |
| Older   | No        |
